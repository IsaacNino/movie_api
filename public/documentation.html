Documentation Page

API Instructions:

This app will guide you through a database of some amazing films.

Users will find information about several movies, directors, and genres.

Users can sign up, update their personal info, and create a list of their favorite movies.

Endpoint Documentation:

Request URL HTTP Method Response
Return a list of ALL movies to the user /movies GET A JSON object containing data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:Title GET A JSON object containing data about a single movie containing the data above
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/:Name GET A JSON object containing data about genre
Return data about a director (bio, birth year, death year) by name /movies/directors/:Name GET A JSON object containing data about director
Allow new users to register /users POST A message informing the user that registration of the account was successful
Allow users to add a movie to their list of favorites (displays a message stating that a movie has been added) /users/:Username/movies/:MovieID POST A message informing the user that movie was added to favourites list
Allow users to update their user info (username) /users/:Username PUT A message informing the user that their information was updated
Allow users to remove a movie from their list of favorites (display text that a movie has been removed—more on this later) /users/:Username/movies/:MovieID DELETE A message informing the user that a movie was removed from their favorites
Allow existing users to deregister (displays text that a user email has been removed) /users/:Username DELETE A message informing the user that deletion of their account was successful
